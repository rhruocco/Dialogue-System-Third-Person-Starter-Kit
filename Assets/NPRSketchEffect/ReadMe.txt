Package include:
1) "NPR Sketch Effect/Sketch" shader used to rendering sketch gameobject.
2) "NPR Sketch Effect/Pencil Contour" shader used to rendering contour outline line, based on camera effect.

NPR Sketch Effect provide a shader solution for realtime pencil sketch-style rendering.
Source code included so you can modify it for your demand.
Demo scene demonstrate all features and usage mode. Play demo scene then you will see sketch effect.
